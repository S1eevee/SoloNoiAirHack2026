# models/

Stores the serialised trained XGBoost model. Contents are git-ignored and regenerated by training.

## Files

### `model.pkl`
Serialised `XGBRegressor` saved with `joblib`. Created by `src/model/train.py` when training completes. Loaded by `src/model/predict.py` for inference.

**Not tracked by git** — each machine trains its own model from local data.

## Model Details
| Parameter | Value |
|---|---|
| Algorithm | XGBoost Regressor |
| Target | `total_pax` per 30-min window |
| Features | `num_flights`, `avg_delay`, `max_delay`, `checkin_desks`, `hour`, `minute`, `day_of_week`, `is_peak_hour`, `time_of_day` |
| n_estimators | 300 |
| max_depth | 6 |
| learning_rate | 0.05 |

## Lifecycle
1. Train: `python3 pipeline.py train` or `POST /forecast/train`
2. Predict: `python3 pipeline.py predict` or `POST /forecast/run`
3. Retrain anytime by uploading new historical data and clicking Train Model in the dashboard
